##  오늘 공부한 내용 (3-17)
- 변수에 대해 알아보자

### 변수
- 컴퓨터는 메모리에 모든 것을 저장, 그런 메모리를 만들기 위해 '변수'란 것이 존재
- 변수 = 메모리 공간의 이름 정하는 것 

### 변수의 저장형태
- 수학 시간에 배웠던 좌우는 같다라는 등호는'=='로 표현을 함 ex: choice == "1":
- 그럼 '='는 무슨 뜻인가? ---> 변수의 값을 저장하라는 의미
- 변수는 정수뿐만 아닌 실수, 문자 etc로 저장 가능함

### 변수의 이름 지정 규칙
- 변수 이름은 무조건 영어 아니면 '_'로 적어야함
- 변수 정할 때 띄어쓰기 금지
- 대문자와 소문자 구별 엄격함
- 변수는 앞글자가 숫자가 오면 안됨

### 변수 사용의 예시
- import turtle
- t = turtle.Turtle()
- t.shape("turtle")
- 'radius' = 100
- t.circle(radius)
- 이렇게 하면 굳이 숫자 하나하나 바꿔주지 않고 반지름 값만 변경하면 됨
